# TransBiolab — Dataset Evaluation Website

TransBiolab 透明物体数据集评测结果展示页面。

## 📁 目录结构

```
.
├── index.html          # 主页面
├── style.css           # 样式表
├── app.js              # 交互逻辑
├── rgb/                # 原始 RGB 图像 
├── sam3/               # SAM3 分割可视化 
├── fp_gt/              # FoundationPose GT 3D bbox 
├── fp_pred/            # FoundationPose 预测 3D bbox 
├── deploy.sh           # GitHub Pages 部署脚本
├── pack.sh             # 打包脚本
└── README.md           # 本文件
```
